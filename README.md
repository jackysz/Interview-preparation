## 记录初衷
在大多数情况下，我们都不能将面试的题目在实际项目中有一个很好的应用，而导致只是背会了相应的知识，而并没有对应的理解和应用。这里我会将自己工作中和学习中碰到的与面试相关的问题相结合，加深理解，能让自己在面试的时候更加从容。

## 常考题目及答案总结
> 由于大多数公司的面试题基本常年不怎么更新，所以可以优先搜索过去的面试题目
